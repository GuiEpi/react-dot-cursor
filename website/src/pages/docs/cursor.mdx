### <a name="zindex-prop"></a>`zIndex` Prop

You can change the z-index of the cursor by modifying the `zIndex` prop. Defaults to `9999`.

```jsx
<Cursor zIndex={1000} />
```

### <a name="theme-prop"></a>`theme` Prop

The theme prop allows you to customize the cursor's appearance and behavior through variants and rules.

```tsx
const theme: CursorTheme = {
  variants: {
    custom: {
      style: {
        borderRadius: '8px',
        width: 'auto',
        height: 'auto',
        padding: '1em',
      },
      content: {
        text: 'Custom Text',
        style: {
          fontSize: '14px',
          fontWeight: 'bold',
        },
      },
    },
  },
  rules: [
    {
      selector: '.custom-element',
      variant: 'custom',
      priority: 1,
    },
  ],
};

<Cursor theme={theme} />;
```

> It is also possible to add several selectors in this way `selector: ['.custom-element1', '.custom-element2']`

### <a name="scale-on-click-prop"></a>`scaleOnClick` Prop

If `false`, do not scale cursor when clicking. Defaults to `true`.

```jsx
<Cursor scaleOnClick={false} />
```

### <a name="default-color-prop"></a>`defaultColor` Prop

Change the default cursor color using any **valid CSS color format**. By default, the cursor uses HSL colors from the <a href="https://ui.shadcn.com/colors" target="_blank" rel="noopener noreferrer">Tailwind CSS color palette</a>:

- Dark mode: `hsl(0 0% 98%)` (zinc-50)
- Light mode: `hsl(240 10% 3.9%)` (zinc-950)

**Examples with explicit CSS color formats:**

```jsx
// HSL colors
<Cursor defaultColor="hsl(349 80% 59%)" />
<Cursor defaultColor="hsl(349, 80%, 59%)" />

// RGB colors
<Cursor defaultColor="rgb(255 64 129)" />
<Cursor defaultColor="rgb(255, 64, 129)" />

// OKLCH colors
<Cursor defaultColor="oklch(0.7 0.15 330)" />

// Hex colors
<Cursor defaultColor="#ff4081" />
<Cursor defaultColor="#f41" />

// Named colors
<Cursor defaultColor="red" />
<Cursor defaultColor="hotpink" />

// Other CSS color functions
<Cursor defaultColor="lab(50% 20 -30)" />
<Cursor defaultColor="lch(50% 50 130)" />
```
